# Uber-
Excited to share my latest Power BI project — an Uber Trip Analysis Dashboard! This dashboard provides a clear and interactive overview of trip performance and key business metrics.
🔍 Key Insights:
✅ 199 Total Bookings
✅ $3.4K Total Booking Value
✅ 22 min Average Trip Time
✅ 4 miles Average Trip Distance
✅ Greenwich Village South – Most Frequent Pickup & Drop-off Point

💡 Features:

Interactive filters by date and city

Visual breakdowns by payment type, trip type (Day/Night), and vehicle type

Insightful KPIs like average booking value, total trip distance, and top locations

This project helped me strengthen my data visualization, DAX, and Power BI storytelling skills, turning raw data into actionable insights.
